My Projects

Giforecast

Giforecast is a web application where users can put in a zip code and it will return the current forecast (Open Weather API) and a corresponding gif (Giphy API).

Technologies I Used

HTML, CSS, JavaScript, jQuery, AJAX

Zach's Fish House

Zach's Fish House (ZFH) is a full CRUD adhering to the MVC file structure and uses all 7 RESTful routes. ZFH allows users to select a fish, see information about each fish, and "buy" the fish if it is still in stock. This app also uses authorization to allow users to make an account and log in. Once users are logged in they are able to "sell" fish and edit or delete the fish that they have listed.

Technologies I Used

JavaScript, CSS, ejs, MongoDB, Mongoose, Semantic UI, Bootstrap, bcrypt

RUMBLE

RUMBLE was a group project where users are shown two babies and they can vote on whichever they think is cuter.

Technologies I Used

React, JSX, CSS, Express, MongoDB, Bootstrap, bcrypt

In Case You Missed It

In Case You Missed It (ICYMI) is an app that allows you to see setlists from previous concerts. Just pick the artist you like and select one of their shows.

Technologies I Used

Ruby on Rails, ERB, PostgreSQL, Bootstrap
